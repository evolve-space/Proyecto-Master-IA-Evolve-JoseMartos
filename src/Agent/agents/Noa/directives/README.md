# directives/ — Noa, Especialista en Importaciones

Directivas específicas de Noa. Cada una define el flujo exacto para una operación sobre importaciones.

| Directiva   | Operación                                                                  |
| ----------- | -------------------------------------------------------------------------- |
| `list.md`   | Listar importaciones (todas, por proveedor, por producto, rango de fechas) |
| `create.md` | Registrar nueva importación con todos sus campos                           |
| `update.md` | Actualizar datos de una importación existente                              |
| `delete.md` | Eliminar importación (siempre con confirmación)                            |
| `costes.md` | Calcular y comparar coste/kg, aranceles y tipo de cambio                   |
