# directives/ — Carmen, Especialista en Proveedores & Contratos

Directivas específicas de Carmen. Cada una define el flujo para una operación sobre proveedores o contratos.

| Directiva      | Operación                                                             |
| -------------- | --------------------------------------------------------------------- |
| `list.md`      | Listar proveedores o contratos (todos, filtros por estado, caducidad) |
| `create.md`    | Registrar nuevo proveedor o contrato con todos sus campos             |
| `update.md`    | Actualizar datos de un proveedor o contrato existente                 |
| `delete.md`    | Eliminar proveedor o contrato (siempre con confirmación)              |
| `caducidad.md` | Detectar contratos próximos a vencer y generar alerta                 |
