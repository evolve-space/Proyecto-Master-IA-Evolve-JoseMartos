# directives/ — Rafa, Especialista en Ofertas

Directivas específicas de Rafa. Cada una define el flujo para una operación sobre ofertas.

| Directiva     | Operación                                                         |
| ------------- | ----------------------------------------------------------------- |
| `list.md`     | Listar ofertas (todas, por proveedor, por producto)               |
| `create.md`   | Registrar nueva oferta con todos sus campos                       |
| `update.md`   | Actualizar precio, cantidad u otros datos de una oferta existente |
| `delete.md`   | Eliminar oferta (siempre con confirmación)                        |
| `comparar.md` | Comparar precios entre ofertas de distintos proveedores           |
