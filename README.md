# VendorFlow AI

Plataforma personalizable de gestión de proveedores y automatización empresarial impulsada por inteligencia artificial.

---

# Descripción del proyecto

Muchas pequeñas y medianas empresas siguen gestionando proveedores mediante procesos manuales, hojas de cálculo o herramientas no adaptadas a sus necesidades reales.

Este proyecto nace como una solución SRM (Supplier Relationship Management) moderna y personalizable orientada a digitalizar la gestión de proveedores, automatizar procesos internos e integrar inteligencia artificial en entornos empresariales reales.

A diferencia de los CRMs tradicionales enfocados en clientes, VendorFlow AI centra su enfoque en la relación con proveedores, optimizando la organización interna, la comunicación y la automatización de procesos de compras y gestión empresarial.

La plataforma ha sido diseñada tomando como referencia un caso de uso real de empresa que necesitaba una solución flexible, moderna y escalable para mejorar sus procesos internos de gestión de proveedores.

# Arquitectura del proyecto

La aplicación sigue una arquitectura modular separada en frontend, backend y servicios de automatización.

```txt
Frontend React
    ↓
Backend Symfony API
    ↓
Servicios de automatización e IA
    ↓
Base de datos MySQL
```

# Ejecución local

## Requisitos
- Node.js
- PHP 8+
- Composer
- MySQL
- Docker (opcional)

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Backend

```bash
cd backend
composer install
symfony server:start
```
